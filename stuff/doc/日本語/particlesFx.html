Source Source (Generator Image) 入力画像の不透明度がThreshold以上のエリアに、均等にパーティクルを生成する。該当フレームに入力画像のセルが無い、又は全面不透明度がThreshold未満の場合は入力画像が無いのと同じ結果になる。 ○
　 　 Threshold 上記生成に用いられるしきい値（0-255） 　
　 　 Multiple Generators in Control Image 透明部分で囲まれた不透明部分（以下「発生源」）が複数ある場合に影響する。
ONのとき： 発生するパーティクルの量が各発生源ごとに均等になる。すなわち、狭い発生源ほどパーティクルの発生密度が濃くなる。
OFFのとき：発生するパーティクルの密度は全ての不透明部分について均等になる。すなわち、狭い発生源ほどパーティクルの量は減る。		 ―
　 　 Use Control Image Gradation 入力画像の不透明度に比例してパーティクルの発生密度を変える。このオプションは上のMultiple Generators in Control Imageと同時には使用できない。 ―
　 　 Center X 入力画像の無い場合の、パーティクル発生エリアの中心位置のX座標。 ―
　 　 Center Y 入力画像の無い場合の、パーティクル発生エリアの中心位置のY座標。 ―
　 　 Width 入力画像の無い場合の、パーティクル発生エリア（四角形）の幅。 ―
　 　 Height 入力画像の無い場合の、パーティクル発生エリア（四角形）の高さ。 ―
　 Particle Generation Starting Frame パーティクルが発生を始めるフレーム時刻。シーンのフレームが基準。負の値も取れる。 ―
　 　 Birth Rate そのフレームで発生するパーティクル数 ―
　 　 Animation Step ParticleのアニメーションをStep値おきにする。※このStep値を2以上にすると、Particle→Animationで指定した絵の変化のStepは設定した値の二乗になる。 ―
　 　 Random Seed このFxので用いられるランダムパラメータのシード値 ―
　 Particle Animation Hold Frame: Sourceカラム（Levelではない）の各セルからランダムに決定する開始フレームのまま固定。
Random Frame: フレームがランダムで変化する。上記AnimationStep値の二乗の周期。
Column: Sourceカラムに入っているセルの順に変化する。最初のフレームからスタート。
Column - Random Start: Sourceカラムに入っているセルの順に変化する。ランダムなフレームからスタート。
Column Swing - Random Start: Sourceカラムに入っているセルの順に変化し、最後までいったら逆再生する。ランダムなフレームからスタート。		 ―

Birth Params Speed Speed パーティクルの初速度 ○
　 　 Linked to Scale パーティクルのスピードの大小をサイズの大小とリンクさせる。奥行きがあるように見えるように、大きいほど早くなる。あまり厳密に比例関係にはないようだ。 ―
　 　 Speed Angle パーティクルの初速の向き。０が下方向、90が右方向 ○
　 　 Use Gradient Angle ONのとき、Speed AngleのControlImageの勾配ベクトルに初速を合わせる。 　
　 Size,Mass& Orientation Size サイズの最大／最小の範囲を指定する。ControlImageが有る場合は、その輝度が高いところほど大きくなる。ControlImageが無い場合はランダム。 ○
　 　 Perspective Distribution ONのとき、かつ、SizeのControlImageが存在する場合、粒子の密度をサイズに反比例させて配置する。海面の波のように、奥行きのある粒子の配置に用いる。このオプションがONのとき、Use Control Image Gradationオプションは無視される。 　
　 　 Use Control Image for the Whole Lifetime ONのとき、各フレーム毎にそのパーティクルの位置のControlImageの輝度に合わせてサイズを変化させる。
OFFのとき、パーティクル発生時のみControlImageを見る。以降は消えるまで同じサイズ。		 ―
　 　 Mass 各パーティクルの質量の最大／最小の範囲を指定する。 質量は下のGravityで与えられるスピードに関わる係数になる。 ―
　 　 Orientation パーティクルの初期の向きの角度の最大／最小の範囲を指定する。０で回転なし、90で右に90度回転。  ○
　 Trail Trail 軌跡を表示させる。Trailで指定したフレーム数かけてフェードアウトする。 ―
　 　 Step 軌跡をStep値おきに表示する。すなわち、軌跡の画像は（Trail）/(Frame)枚表示されることになる。 ―
　 Lifetime Lifetime パーティクルの寿命(フレーム数)の最大／最小の範囲を指定する。寿命はパーティクルが現れ始めてから完全に消えるまでのフレーム長。 ○
　 　 Use Column Duration for Lifetime ONのとき、パーティクルの寿命がSourceColumnの長さになる。 ―
　 Top Layer Top Layer パーティクルの重ね順の優先順位を指定する。 ―
Page Category Param Name Description Control Image
Environment Gravity Gravity パーティクルに与える重力（＝加速度）の値を指定する。負の値で逆方向になる。ControlImageを入力した場合は下のGravityAngleの値は用いられず、入力画像の輝度の勾配（Gradient）が重力となる。明るい方に引き寄せられる。 ○
　 　 Gravity Angle 重力の方向を指定する。０で下向き。90で左向きに落ちていく。 ―
　 Friction Friction パーティクルの運動を止めるような摩擦の力の値を指定する。入力画像の輝度が最大値の部分で最大の摩擦力となる。入力画像が無ければ無視される。 ○
　 Wind Wind Intensity パーティクルを一様に動かす風の速さを指定する。加速度ではなく、速度に足される。要は初速度（Birth Param→Speed）と同じ意味と思われる。 ―
　 　 Wind Angle 風の向きを指定する。0で真上、90で右方向。 ―
　 Scattering Horizontal パーティクルのランダムな動きのパラメータを指定する。パーティクルにランダムに動きを与える水平方向のスピード値の範囲を指定する。 ○
　 　 Vertical パーティクルにランダムに動きを与える垂直方向のスピード値の範囲を指定する。 ○
　 　 Swing Mode Random： 各フレーム毎にHorizontal/Vertical値を変える。ただし、下のSwing値で指定されたフレーム数の間は、Horizontal/Vertical値の正負の符号は変わらない。
Smooth： 下のSwing値で指定されたフレーム数の間、Horizontal/Vertical値は不変。		 ―
　 　 Swing SwingModeがSmoothのとき、Horizontal/Vertical値を再設定するフレーム間隔の最小/最大値を指定する。 ―
Page Category Param Name Description Control Image
Animation Rotation Rotation Speed 各パーティクルの回転スピードを指定する。1フレームあたりの回転角（単位degree） ―
　 　 Extra Speed 各パーティクルにランダムで追加される角度の範囲を指定する。 ―
　 　 Swing Mode Random: 各フレーム毎にExtraSpeedを計算しなおす。
Smooth: 下のRotationSwingで指定されたフレーム数の間、同じExtraSpeed値で回転する。		 ―
　 　 Rotation Swing SwingModeがSmoothのとき、ExtraSpeed値を再設定するフレーム間隔の最小/最大値を指定する。 ―
　 　 Follow Particles Movement ONのとき、パーティクルの回転角を進行方向に沿わせる。右水平方向に進む時、回転角は０。 この回転は上記のRotationと重ねがけできる。 ―
　 Opacity Opacity パーティクルの不透明度の最小/最大値。最小値から最大値までフェードインして、最小値までフェードアウトするようになる。
ControlImageを入力すると、入力画像の輝度値に応じてそのフレーム時点でのパーティクルの不透明度が決まる。真っ黒のときに最小値の不透明度になる。不透明度は毎フレーム更新される。
ControlImageによるOpacityは、下のフェードイン/アウトと重ねがけすることができる。		 ○
　 　 Fade-in Frames フェードインにかかるフレーム数 ―
　 　 fade-out Frames フェードアウトにかかるフレーム数 ―
　 　 Trail Opacity BirthParam→TrailでTrailを指定したとき、その軌跡のパーティクルの不透明度の最小/最大値を決める。 ―
　 Size Increase Size Intensity 各フレーム毎にランダムで変化するサイズの増減値（％）の最小/最大値を指定する。 ○

Colors Birth Color Birth Color パーティクル発生時の色を決定する。
それぞれのパーティクルが生成されるときに、スペクトルで指定された色の範囲からランダムに選ばれる。
ControlImageを指定した場合には、入力画像の（Premultiplyされた）RGB値がパーティクルの色に使用される		 ○
　 　 Birth Spread 　 ―
　 　 Birth Intensity Birth Colorの影響度（％） ―
　 　 Pick Control Image's Color for Every Frame 毎フレーム、現在のパーティクルの位置にあるControlImageのピクセルを参照し、パーティクルの色を変える ―
　 Fade-in Color Fade-in Color パーティクルがフェードインしていく色を決定する。
スペクトルで指定された色の範囲からランダムに選ばれる。
ControlImageを指定した場合には、入力画像の（Premultiplyされた）RGB値がパーティクルの色に使用される		 ○
　 　 Fade-in Spread 　 ―
　 　 Frame Range パーティクルの発生から数えてFade-in Colorに至るまでのフレーム数 ―
　 　 Fade-in Intensity Fade-in Colorの影響度（％） ―
　 Fade-out Color Fade-out Color パーティクルがフェードアウトしていく色を決定する。
スペクトルで指定された色の範囲からランダムに選ばれる。
ControlImageを指定した場合には、入力画像の（Premultiplyされた）RGB値がパーティクルの色に使用される		 ○
　 　 Fade-out Spread 　 ―
　 　 Frame Range パーティクルの消滅から逆に数えてFade-out Colorへ変化し始めるフレーム数 ―
　 　 Fade-out Intensity Fade-out Colorの影響度（％） ―
