Motion Blur (Desenfoque por movimiento) Iwa

● Descripción

Este efecto permite generar un desenfoque causado por el movimiento.
Especificando valores de Shutter Start y Shutter End se aplicará un desenfoque de acuerdo a
la trayectoria, del objeto durante ese tiempo.
Tal como sucede en el efecto Bokeh Iwa, los valores RVA son convertidos en un valor de
exposición antes de ser desenfocados, por lo tanto aunque la partes más claras se encuentren
desenfocadas, no habrá una pérdida de brillo en las mismas.

● Entradas

  • Source (Origen) : Permite conectar la imagen a ser procesada.

● Opciones

  • Reference Object (Objeto de referencia), Index (Número) : Permite especificar el objeto
    del cual extraer la información de movimiento.
    La opción "Own Motion" (Movimiento propio) permite seguir el movimiento de la columna
    conectada a Source.
  • Shutter Start (Apertura obturador) : Permite especificar qué tanto antes del fotograma
    actual deberá abrirse el obturador.
    Es expresado en fotogramas. Cuanto mayor sea el valor, más se extenderá el desenfoque.
  • Start Value (Valor apertura) : Permite especificar la intensidad del desenfoque en el
    momento en que se abre el obturador.
  • Start Curve (Curva apertura) : Permite especificar cómo aplicar el desenfoque, desde el
    fotograma actual, hasta el momento en que se abre el obturador.
    El valor predefinido es 1, con el cual el desenfoque cambiará de forma lineal.
  • Shutter End (Cierre obturador) : Permite especificar qué tanto después del fotograma
    actual deberá cerrarse el obturador.
    Es expresado en fotogramas. Cuanto mayor sea el valor, más se extenderá el desenfoque.
  • End Value (Valor cierre) : Permite especificar la intensidad del desenfoque en el momento
    en que se cierra el obturador.
  • End Curve (Curva cierre) : Permite especificar cómo aplicar el desenfoque, desde el
    fotograma actual, hasta el momento en que se cierra el obturador. El valor predefinido es 1,
    con el cual el desenfoque cambiará de forma lineal.
  • Trace Resolution (Resolución seguimiento) : Permite definir qué tan detallado será el
    seguimiento de la trayectoria de movimiento del objeto, mientras el obturador se encuentre
    abierto.
    Cuando sea 1, la trayectoria será una línea recta, aunque el objeto se encuentre moviéndose
    a lo largo de una curva.
  • Hardness (Dureza) : Valor de gama de la película. Es usado para convertir los valores RVA y
    la cantidad de exposición. Cuanto mayor sea el valor, más claro será el efecto.
  • Afterimage Mode (Modo estela) : Cuando se encuentre activo, en vez de un desenfoque suave,
    se dibujará una estela compuesta por una serie de repeticiones de la imagen original, en la
    posiciones calculadas según la trayectoria, y respetando el parámetro "Trace Resolution".
