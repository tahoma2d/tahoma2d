Perspective Distort Fx Iwa

● Overview

Allows to generate a motion parallax movement effect on the Source image, where things in
front move faster than things in the back.
The image width is stretched to the sides, to transform it into a trapezoid, but it will not
be stretched vertically.
In other words, the contraction in the vertical direction caused by depth, should be
incorporated beforehand in the image.
Also, if you want to apply this effect to a moving background, the movement of the background
must be collapsed into a SubXsheet.

● Input Port

  • Source : Connect the input image.

● Parameters

  • Vanishing Point : The center point for the deformation. In the case of perspective
    projection, this is the vanishing point.
  • Anchor Point : Defines the vertical position that will be the basis for the deformation.
    Pixels above this point will not be drawn (become transparent).
    It also defines the horizon line for the deformed image, maintaining the original image width.
  • Precision : Since the deformation is carried out only by expansion, the resolution will
    suffer. In order to minimize resolution deterioration, the original image resolution is
    multiplied by this value before being deformed.
    There is no point in making Precision higher than the value obtained by dividing the
    "Capture resolution of the background" by the "Output resolution".
