MotionBlurFx Iwa

● Overview

This effect generates motion blur.
If you specify "Time during which the shutter is open before and after the current frame",
the image will be blurred according to the object's path.
As with iwa_BokehFx, the RGB values are converted into an exposure value before blurring,
Therefore, even if the highlight part is blurred, the brightness is not impaired.

● Input Port

  • Source : Connect the input image.

● Parameters

  • Reference Object, Index : Specify the object to follow the movement.
    When "Own Motion" is specified, the column movement of the input image is tracked.
  • Shutter Start : Specify how long before the current frame the shutter should be opened.
    The unit is in frames. The higher the value, the bigger the blur.
  • Start Value : Specify the strength of the blur when the shutter is opened.
  • Start Curve : Specify how to apply the blur, from the time the shutter is opened up to
    the current frame. The default value is 1, with it the blur changes linearly.
  • Shutter End : Specify how long after the current frame the shutter should be closed.
    The unit is in frames. The higher the value, the bigger the blur.
  • End Value : Specify the strength of the blur when the shutter is closed.
  • End Curve : Specify how to apply the blur, from the current frame up to the time
    the shutter is closed. The default value is 1, with it the blur changes linearly.
  • Trace Resolution : Defines how detailed will be the tracking of the object's trajectory,
    while the shutter is open.
    At 1, the blur trajectory will be a straight line, even if the object is moving along a curve.
  • Hardness : Gamma value of the film. It is used to convert the RGB values and exposure
    amount. The higher the value, the brighter the effect.
  • Afterimage Mode : When active, instead of a smooth blur, an afterimage is drawn at the
    position traced, using the detail set with the "Trace Resolution" parameter.
